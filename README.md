## High level concept

Red social para crear, compartir y comparar listas de películas, series y anime.

## Funcionalidades principales

-- Top de películas y series y series preferidas

-- Lista de películas y series que he visto

-- Lista de películas y series que quiero ver

-- Perfil de usuario

-- Lista de Homies (amigos)

-- Base de datos actualizada con películas y series

-- Registro e inicio de sesión con autenticación JWT


## Stack tecnológico

FrontEnd | HTML+CSS+JS con Bootstrap
Backend | Java 21 + Spring Boot 3.2
Base de datos | PostgreSQL + Neon.tech
Autenticación | ---Spring Security + JWT (jjwt)
Deploy Frontend | Vercel 
Deploy backend | Railway 
Mockup y diseño | Whiteboard + Adobe XD
Gestion | Jira + Discord + Google Drive
