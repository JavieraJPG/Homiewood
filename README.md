# Homiwood

Red social para crear, compartir y comparar listas de películas, series y anime.

## Tecnologías

### Backend
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- TMDB API
- Jikan API

### Frontend
- Pendiente
